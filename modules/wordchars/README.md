wordchars  module
=================


how we work
-----------

- enable set WORDCHARS for bindkey settings

```
# by default: export WORDCHARS='*?_-.[]~=/&;!#$%^(){}<>'
##  we take out the slash"/", period".", angle brackets"<>", dash here.
export WORDCHARS='*?_[]~=&;!#$%^(){}'
```
